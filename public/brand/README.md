# Brand marks

The "bi" monogram in **Poppins Bold**, monochrome white on a tile. Generated
from `inode/social/brand/source/Poppins-Bold.ttf` via opentype.js.

`mark.svg` is the live website mark (lowercase "bi", rounded, navy). The full
variant library:

| Folder | Letterform |
| ------ | ---------- |
| `bi/`  | lowercase "bi" |
| `Bi/`  | capital "Bi"   |

Each folder has, as both `.svg` (vector) and `.png` (1024px, for uploads that
don't accept SVG):

- `rounded-*` — rounded-square tile, for the website / app icons
- `square-*` — no-radius square, for profile pictures (LinkedIn, GitHub, etc.
  that crop to a circle)
- `*-navy` — navy `#003554` tile
- `*-coral` — coral/red `#a80000` tile

`icon.svg` is the original "b." mark, kept for reference.
