# "bi" logo — sample (not live)

Proposed alternate mark: the original logo with the `.` turned into an `i`, so it
reads "bi" instead of "b." (and looks distinct from Booking.com). Same tile,
same colors, same letterforms as the live logo. The blue accent that was the
period now sits as the dot on the `i`.

These are samples to compare later. The live logo is unchanged — it still lives
at `public/brand/mark.svg`, `public/brand/icon.svg`, and `src/app/icon.svg`.

| ---------- | ---------- | ------- | ------------ |
| `mark.svg` | navy       | white   | sky `#38bdf8` |
| `icon.svg` | white      | navy    | blue `#00a6fb` |

`.png` files are 512px previews of each. To adopt this later, copy these over
the live files above.
